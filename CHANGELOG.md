# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-01-14

### Added
- Initial release of AI Incident Commander
- FastAPI backend with REST API and WebSocket support
- React frontend with real-time dashboard
- LangChain AI Commander for incident analysis
- Vector database (FAISS) for RAG-based knowledge retrieval
- Async job queue for background incident analysis
- Prometheus metrics and observability
- Rate limiting and security features
- Docker and Docker Compose support
- Comprehensive documentation (API, Architecture, Deployment)
- Seed data script with demo incidents
- Real-time incident rooms with WebSocket
- Timeline tracking and action suggestions
- Event ingestion for logs, metrics, and alerts

### Features
- Create and manage incidents
- Automatic severity classification
- AI-powered root cause analysis
- Similar incident detection

- Real-time collaboration
- Metrics visualization
- Full observability stack
